Sharik is an app for sending files to almost any device. It works with Wi-Fi connection or Tethering (Mobile Hotspot). No internet connection needed.

## Platforms
- andoid
- windows (a bit buggy)
- linux
- macos (probably)
- ios (maybe soon)
